Projects Notable pieces of code that keep me busy outside of the workplace

Gameday Roster

Finding out who's in or out for the next pickup football game, soccer match, or poker night can be a tedious task. While still in early development, Gameday Roster is aimed at providing a solution to this issue.

Magnum CI

When Resque was released I was looking for a fun project to try it out on. It just so happened that we weren't completely happy with our Continuous Integration (CI) server at work, so I started Magnum CI as an attempt to create a simple CI server.

Kegbot

Kegbot is a free, open-source project to turn your beer kegerator into a computerized drink tracker. My goal in the project is to help streamline the setup and installation of the hardware/software making it as simple as possible for anyone to setup their own kegbot.

Double Auth Engine

I utilize Authlogic and Declarative Authorization on all Rails projects. In an effort avoid writing the same boilerplate code for each new project, I decided to leverage the engine functionality in Rails and created Double Auth Engine.
